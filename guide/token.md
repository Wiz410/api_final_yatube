# JWT
Отправте `POST` запрос на адрес: `http://127.0.0.1:8000/api/v1/jwt/create/` с вашим логином и паролем.

```json
{
    "username": "логин",
    "password": "пороль"
}
```

В поле `access` ваш токен, а в `refresh` токен для обновления.
Время действия токена **10 дней**.

```json
{
    "refresh": "токен",
    "access": "токен"
}
```
